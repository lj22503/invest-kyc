## 🚀 部署流程

### 快速部署（3 步）

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署
cd /home/admin/.openclaw/workspace/projects/mangofolio-h5
vercel --prod
```

### 绑定域名

**Vercel 子域名**（推荐）：
- `mangofolio.vercel.app`
- 在 Vercel Dashboard → Settings → Domains 添加

**自定义域名**：
- `test.mangofolio.com`
- DNS 配置：`CNAME cname.vercel-dns.com`

## 📝 使用说明

### 用户体验流程

1. **打开测试页** → `https://mangofolio.vercel.app`
2. **回答 10 道题** → A/B/C/D 选择
3. **查看结果** → 本命宠物 + 投资人格标签
4. **阅读风险提示** → 合规声明
5. **下载技能** → 一键安装投资宠物
6. **加入社群** → 扫码进群
7. **分享结果** → 分享到朋友圈

### 运营配置流程

1. **配置二维码** → 参考 `SELF_OPERATION.md`
2. **自定义文案** → 编辑 `public/index.html`
3. **部署更新** → `vercel --prod`
4. **查看数据** → Vercel Analytics / Google Analytics
