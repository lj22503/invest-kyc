## 🔧 代码优化建议

### 1. 题目数量扩展

**当前**：10 道题  
**建议**：20 道题（更准确的人格评估）

**新增题目方向**：
- 投资知识水平测试
- 风险承受能力量化
- 决策风格评估
- 情绪稳定性测试
- 时间偏好测试

### 2. 匹配算法优化

**当前**：简单累加（每选一次 +2 分）  
**建议**：五维评分 + 加权计算

**五维模型**：
1. 风险承受力（0-100）
2. 知识水平（0-100）
3. 决策风格（理性/感性）
4. 情绪稳定性（0-100）
5. 时间偏好（短期/长期）

### 3. 结果页优化

**当前**：文字描述 + Top 3  
**建议**：增加可视化元素

**新增内容**：
- 五维雷达图
- 投资性格详细分析
- 推荐学习路径
- 个性化建议

### 4. 数据持久化

```javascript
// 保存测试结果
localStorage.setItem('invest-kyc-result', JSON.stringify({
  petId: bestPetId,
  scores: petScores,
  date: new Date().toISOString()
}));
```
