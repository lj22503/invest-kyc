**燃冰** 一人 CEO 2026 年 4 月 14 日

从今天起，投资这件事，有人陪你了。

不是理财师，不是机器人，是一只懂你的宠物。

---

## 一、为什么做这件事？

最近有两件事挺火的。

**一是把人格"蒸馏"成 Skill**。设计师的审美、工程师的代码习惯、销售的话术，都能封装成 AI 技能。人会离职，但 Skill 永存。

**二是 SBTI 测试**。为什么让人上瘾？因为它用一种有趣又自嘲的方式，给了你一个理解自己的框架。"你的人格是 SOLO，我哭了，我怎么会是孤儿"——很高效也不准，不过有种表演的艺术。

我最近在搞金融中的"顾"的部分，就在想：

> 投资者行为测试和行为背后的人格化 Skill，能不能合到一起？

不是冷冰冰的"保守型/平衡型/进取型"，而是有性格、有故事的形象。

比如"你是🐿️ 松果型投资者"——爱囤积，谨慎，但能坚持。

是不是比"C3 型投资者"有意思多了？

## 二、一个扎心的事实

因为我在金融行业待得越久，越发现一个扎心的事实：

**资产管理和财富管理，差异很大。**

资产端我们讲究的事情，在用户端往往是反方向：

- 你知道长期持有好，但市场大跌那天，你还是忍不住打开了卖出页面。
- 你知道别追涨杀跌，但看到别人赚钱，你还是怕错过。
- 你知道要分散配置，但那只基金涨得太好了，你还是想重仓赌一把。

这些不是知识问题，是人性问题。

行为金融学研究了 50 年，发现了一堆偏差：损失厌恶、从众心理、过度自信、现时偏好……

知道这些有用吗？

有点用。但知道归知道，操作归操作。

就像减肥。你知道要少吃多动，但看到蛋糕还是忍不住。

**反人性的事，靠意志力撑不久。**

那怎么办？

传统的投资者教育，是上课、读书、听讲座。告诉你"要长期""别追涨杀跌""要分散"。

但道理都懂，还是做不好投资和理财。

问题出在哪？

我觉得，大家都在跟风，没有找到符合自己特征的投资方法。

也有一些做得好的，比如有知有行的投教面向中产，贴近生活场景，跟人的距离感就挺好的。

但对于更普通的用户，能获得的符合自己的方法太少。我体验了一圈蚂小财们，发现都是制式化的模板，符合平台的设定去做了一些框架，有些分析不够个性化，而且带有强烈的销售视角。

而纯粹去问 AI，又没有章法，并且数据上问题很大。

我就想，能不能用一种有趣的方式，让投资这件事变得没那么难？

不是教育你，而是陪伴你。

不是说教你，而是理解你。

不是让你变成别人去投资，而是帮你获取更适合自己的投资方法。

于是我就想画出每类人的投资特点，然后做一个注入了使用者需求、有灵魂的一只宠物 Agent。

跟着你一起成长。不像理财师，也不像现在的平台小助手，是你专属的服务。

## 三、为什么是宠物？

先说一个故事。

我之前当产品经理时，做过很多投资工具，功能挺全的：行情、指数、基金诊断、资产诊断，但都很零散。

上线需要推，推完就完了，后续得用户自己去找去用。活跃度和留存难搞。

用户调研，有人说："功能挺好，但不知道怎么用，需要主动去找，想用的时候太麻烦了。"

**需要主动找。**

这三个字点醒了我。

投资本来就是反人性的，工具还需要主动用，那怎么经营得下去？

市场跌了，你本来就慌，打开 APP 一片冷冰冰的数字和图表。

市场涨了，你本来就飘，打开 APP 还是冷冰冰的数字和图表。

没有人关心你，只有数字告诉你涨跌。

那什么是有温度的？

我想到了宠物。

宠物不会评判你，不会说"你怎么又操作了"，它只会说"我知道你担心，我陪你"。

宠物不会离开你，不会今天热情明天冷淡，它一直都在。

宠物不会要求你变成别人，它接受你本来的样子。

**宠物不是工具，是伙伴。**

所以，我做了 12 只投资宠物去适配不同的投资者。

不是换个皮肤的机器人，是真的有"人格"的伙伴。

## 五、宠物能干什么？

先说一个真实场景。

**2024 年 10 月 8 日，A 股大跌 6.8%。**

那天下午，我朋友给我发消息："我全卖了，亏 20%。"

我问："为什么？"

他说："受不了了，每天打开账户都是绿的。"

我说："那你现在什么感觉？"

他说："空落落的。好像做错了什么。"

三天后，市场反弹了 4%。他没买回来。

再三天后，市场涨回去了。他彻底踏空。

这个故事里，缺了什么？

缺了一个会在他想卖出时，拦住他的东西。

**宠物就是那个东西。**

---

### 场景 1：早上 8:30，你刚醒

```
🐿️ 松果：早上好！今天也是存坚果的一天！☀️

📊 今日简报：
• 沪深 300 +0.5%
• 你的持仓 +0.8%
• 定投日！记得打卡哦~
```

这不是推送。这是你的宠物在跟你打招呼。

就像你养的那只狗，每天早上会蹭你的腿。

它不是在要吃的，它是在说："我在，你今天也要加油。"

---

### 场景 2：下午 2:30，市场大跌 3%

你打开账户，一片绿。

心跳加速，手心出汗，手指已经放在了"卖出"按钮上。

这时候，宠物说话了：

```
🐿️ 松果：主人，今天跌了 3.2%... 我知道你有点担心。

但历史上沪深 300 跌幅>3% 有 287 次，
其中 263 次在 3 个月内涨回来了（91.6%）。

你已经坚持定投 182 天了，
现在正是积累便宜筹码的时候！

要一起看看历史数据吗？📈
```

你不是被说服的，你是被数据说服的。

91.6% 的概率，182 天的坚持，3.2% 的跌幅。

这三个数字放在一起，你的手从"卖出"按钮上移开了。

**这就是宠物的作用：在你最脆弱的时候，给你一个理性的理由。**

---

### 场景 3：晚上 9:00，你看到别人赚钱了

朋友圈有人在晒收益："这只基一周涨了 15%！"

你心动了。打开 APP，准备重仓买入。

宠物又说话了：

```
🐿️ 松果：主人，我注意到你想买入这只基金...

能告诉我原因吗？
A. 深入研究过  B. 相信基金经理  C. 怕错过  D. 就是手痒

如果是 C 或 D，我建议先冷静一下~
历史上 80% 的「追涨买入」，3 个月内都会回调。

要不我们先聊聊？🫂
```

你停住了。

不是因为宠物拦着你，是因为你发现自己选的是 C 或 D。

**承认自己是在"怕错过"，比盲目追涨，需要更大的勇气。**

宠物给了你这个勇气。

这时候你才发现：

收益 15.3% 当然开心，但更开心的是——

你不再是那个会在大跌时卖出的人了。

你不再是那个看到别人赚钱就追的人了。

你不再是那个每天看净值、情绪被市场牵着走的人了。

**宠物记录的不只是收益，还有你变成了一个什么样的人。**

它不是工具。工具是你用它，它不会主动找你。

- 你难过时，它陪你扛着
- 你想犯错时，它拦住你
- 你成长时，它为你鼓掌

它不会离开你，不会今天热情明天冷淡，它一直都在。

它不会要求你变成别人，它接受你本来的样子。

它不会评判你，它只会说："我知道你担心，我陪你。"

**宠物不是让你变成更好的投资者，而是帮你成为更好的自己。**

## 八、技术上怎么实现？

简单说，这整个有趣的投资者教育，我做成了本地运行的 AI Skill。

**四层架构**：

```
┌─────────────────────────────────────┐
│  用户层：H5 测试 + 结果页            │
│  → 找到你的本命宠物                 │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  宠物层：12 只宠物配置 + 心跳引擎    │
│  → 主动触发、人格化对话              │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  大师层：18 位投资大师建议           │
│  → 复杂问题召唤大师                 │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  数据层：市场数据 + 合规检查         │
│  → 免费 API + 本地缓存               │
└─────────────────────────────────────┘
```

**数据流**：

1. **用户在 H5 做测试** → 得到本命宠物推荐
2. **复制链接，发给小龙虾** → 安装 investment-buddy-pet 技能
3. **宠物激活** → 心跳引擎启动，每天主动触发
4. **市场数据获取** → 新浪财经/天天基金免费 API
5. **宠物生成消息** → 基于人格配置 + 市场数据
6. **合规检查** → 每条消息经过合规过滤器
7. **发送给用户** → 飞书/微信/短信等多渠道

**数据接口规范**：

| 数据类型 | 来源 | 更新频率 | 缓存策略 |
|---------|------|---------|---------|
| 指数行情 | 新浪财经 | 实时 | 1 分钟 |
| 基金净值 | 天天基金 | 每日 | 24 小时 |
| 个股行情 | 腾讯财经 | 实时 | 1 分钟 |
| 宏观数据 | 国家统计局 | 月度 | 7 天 |
| 用户持仓 | 用户手动导入 | 手动 | 5 分钟 |

**合规检查**：

每条消息必须通过合规过滤器：
- ❌ 不推荐具体基金/股票
- ❌ 不承诺收益或保证赚钱
- ❌ 不提供市场择时建议
- ✅ 必须有风险提示
- ✅ 必须有数据免责声明

**代码开源**：

- GitHub：https://github.com/lj22503/investment-buddy-pet
- 数据不出设备
- 代码透明可审查
- 欢迎贡献代码和想法

**合规优先于功能。**

因为金融类产品，透明才能信任。
